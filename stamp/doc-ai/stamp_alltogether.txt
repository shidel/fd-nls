1. Funcionalidades
------------------

- alterar data/hora de ficheiros
- alterar atributos de ficheiros e diretórios
- comparar data/hora e atributos de ficheiros e diretórios
- opções de data/hora permitem obter valores da data/hora atual e do ficheiro
  simultaneamente com números explícitos
- comparação complexa de data/hora com a data/hora atual, tal como com o
  ficheiro
- data/hora em formatos que respeitam as definições de PAÍS (COUNTRY)
- permitidos caminhos UNC e NetWare
- múltiplos alvos numa única linha de comandos
- caracteres universais (wildcards) estendidos: os nomes podem conter qualquer
  número de '*', bem como '?'; "nome" significa "nome.", mas "*" significa
  "*.*"
- deteção automática do carácter de opção ('/' ou '-')
- opção para incluir nomes de diretórios na pesquisa por caracteres universais
- opção para processar ficheiros e diretórios em todos os subdiretórios do
  alvo
- opção para processar ficheiro com lista de alvos; a falta de argumento
  significa entrada padrão (stdin)


2. Níveis de erro (Errorlevels)
-------------------------------

0   - definição ou comparação de data/hora e atributos bem-sucedida
1   - comparação (uma delas) mal-sucedida
2   - erro no acesso ao ficheiro ou na definição de data/hora/atributos
128 - opção inválida ou duplicada
129 - opções incompatíveis, nenhum alvo ou argumentos inválidos
130 - erro interno (memória insuficiente ou diretório demasiado profundo)


3. Tarefas pendentes (Todo)
---------------------------

- mostrar listas de ficheiros processados
- comparar/copiar data/hora/atributos de ficheiro para ficheiros
- incremento/decremento para campos de data/hora
- comparação de maior/menor para atributos

1. Funzionalità
----------------

- modifica data/ora per i file
- modifica attributi per file e directory
- confronta data/ora e attributi per file e directory
- le opzioni data/ora consentono di ottenere valori dalla data/ora corrente
  e da quella del file simultaneamente con numeri espliciti
- confronto complesso di data/ora con la data/ora corrente, così come con
  il file
- data/ora in formati conformi alle impostazioni COUNTRY
- consentiti percorsi UNC e NetWare
- bersagli multipli in una singola riga di comando
- caratteri jolly (wildcards) estesi: i nomi possono contenere qualsiasi
  numero di '*', così come '?'; "nome" significa "nome.", ma "*" significa
  "*.*"
- rilevamento automatico del carattere opzione ('/' o '-')
- opzione per includere i nomi delle directory nella ricerca con caratteri
  jolly
- opzione per elaborare file e directory in tutte le sottodirectory del
  bersaglio
- opzione per elaborare un file con un elenco di bersagli; l'argomento
  mancante indica lo standard input (stdin)


2. Livelli di errore (Errorlevels)
----------------------------------

0   - impostazione o confronto di data/ora e attributi riuscito
1   - il confronto (uno di essi) non è riuscito
2   - errore di accesso al file o nell'impostazione di data/ora/attributi
      del file
128 - opzione non valida o duplicata
129 - opzioni incompatibili, nessun bersaglio o argomenti non validi
130 - errore interno (memoria insufficiente o directory troppo profonda)


3. Cose da fare (Todo)
----------------------

- mostra elenchi di file elaborati
- confronta/copia data/ora/attributi da file a file
- incremento/decremento per i campi data/ora
- confronto maggiore/minore per gli attributi

1. Mogelijkheden
----------------

- wijzig datum/tijd voor bestanden
- wijzig attributen voor bestanden en mappen
- vergelijk datum/tijd en attributen voor bestanden en mappen
- datum/tijd-opties maken het mogelijk om gelijktijdig waarden van de huidige
  en de bestandsdatum/-tijd te verkrijgen met expliciete getallen
- complexe datum/tijd-vergelijking met huidige tijd, evenals met bestand
- datum/tijd in formaten die voldoen aan COUNTRY-instellingen
- UNC- en NetWare-paden toegestaan
- meerdere doelen in één commandoregel
- uitgebreide jokertekens (wildcards): namen kunnen elk aantal '*' bevatten,
  evenals '?'; "naam" betekent "naam.", maar "*" betekent "*.*"
- automatische detectie van optie-teken ('/' of '-')
- optie om mapnamen op te nemen in zoekopdrachten met jokertekens
- optie om bestanden en mappen in alle submappen van het doel te verwerken
- optie om bestand met lijst van doelen te verwerken; ontbrekend argument
  betekent standaard invoer (stdin)


2. Foutniveaus (Errorlevels)
----------------------------

0   - instellen of vergelijken van datum/tijd en attributen succesvol
1   - vergelijking (één van de) is niet succesvol
2   - fout bij bestandstoegang of instellen van bestandsdatum/-tijd/
    -attributen
128 - ongeldige of gedupliceerde optie
129 - incompatibele opties, geen doelen of ongeldige argumenten
130 - interne fout (onvoldoende geheugen of mapstructuur te diep)


3. Te doen (Todo)
-----------------

- tonen van lijsten met verwerkte bestanden
- vergelijken/kopiëren datum/tijd/attributen van bestand-naar-bestanden
- verhogen/verlagen voor datum/tijd-velden
- groter dan/kleiner dan vergelijking voor attributen

1. Funktioner
-------------

- ändra datum/tid för filer
- ändra attribut för filer och kataloger
- jämföra datum/tid och attribut för filer och kataloger
- datum/tid-alternativ gör det möjligt att hämta värden från nuvarande och
  filens datum/tid samtidigt med explicita tal
- komplex datum/tid-jämförelse med nuvarande tid, likaså med fil
- datum/tid i format som följer COUNTRY-inställningarna
- UNC- och NetWare-sökvägar tillåtna
- flera mål i en enda kommandorad
- utökade jokertecken (wildcards): namn kan innehålla valfritt antal '*',
  liksom '?'; "namn" betyder "namn.", men "*" betyder "*.*"
- automatisk identifiering av alternativtecken ('/' eller '-')
- alternativ för att inkludera katalognamn i sökning med jokertecken
- alternativ för att bearbeta filer och kataloger i alla målets underkataloger
- alternativ för att bearbeta fil med en lista över mål; saknat argument
  innebär standardinmatning (stdin)


2. Felnivåer (Errorlevels)
--------------------------

0   - inställning eller jämförelse av datum/tid och attribut lyckades
1   - jämförelsen (en av dem) misslyckades
2   - fel vid filåtkomst eller inställning av fildatum/tid/attribut
128 - ogiltigt eller duplicerat alternativ
129 - inkompatibla alternativ, inga mål eller ogiltiga argument
130 - internt fel (för lite minne eller för djup katalogstruktur)


3. Att göra (Todo)
------------------

- visa listor över bearbetade filer
- jämföra/kopiera datum/tid/attribut från fil-till-filer
- öka/minska fält för datum/tid
- jämförelse av större/mindre för attribut

EDLIN 2.14

Das Programm edlin ist ein kleiner Zeileneditor, der für FreeDOS als
funktionaler Klon des alten MS-DOS-Programms edlin geschrieben wurde. Es
unterscheidet sich vom MS-edlin erstens dadurch, dass es freie Software ist,
und zweitens, dass die Benutzeroberfläche an einigen Stellen leicht
abweicht. Der Grund für diesen Unterschied liegt darin, dass der Benutzer
keine Steuerzeichen eingeben muss, die durch die Syntax von MS-edlin
vorgeschrieben sind.

AUFRUF VON EDLIN

Das Programm edlin wird entweder allein aufgerufen:

  edlin

oder mit einem Dateinamen, der initial eingelesen wird:

  edlin datei

INTERNE BEFEHLE VON EDLIN

In den folgenden Befehlslisten stellen eckige Klammern optionale Konstrukte
dar. Das Symbol # steht für eine Zeilennummer, die entweder eine Ziffernfolge,
ein Punkt für die aktuelle Zeile, ein Dollarzeichen ($) für die letzte Zeile
in der Datei oder eine Zeilennummer ist, die zu einer anderen Zeilennummer
addiert oder davon subtrahiert wird, sodass

  .,.+1p

die aktuelle Zeile und die darauffolgende Zeile ausgibt.

Ein Dollarzeichen in den folgenden Listen stellt eine Zeichenkette dar.
Diese können entweder in einfache oder doppelte Anführungszeichen gesetzt
werden und die folgenden Escape-Sequenzen enthalten:

  \a - Alarm (alert)
  \b - Rücktaste (backspace)
  \e - Escape
  \f - Seitenvorschub (formfeed)
  \t - Horizontaler Tabulator
  \v - Vertikaler Tabulator
  \" - Doppeltes Anführungszeichen
  \' - Einfaches Anführungszeichen
  \. - Punkt
  \\ - Backslash
  \xXX wobei X eine Hexadezimalziffer (0-9, a-f, A-F) ist - Hex-Konstante
  \dddd wobei d eine Dezimalziffer (0-9) ist - Dezimal-Konstante
  \OOO wobei O eine Oktalziffer (0-7) ist - Oktal-Konstante
  \^C - Steuerzeichen (control character)

Das Programm edlin unterstützt die folgende Syntax:

# - EINE EINZELNE ZEILE BEARBEITEN

edlin gibt die durch die Nummer angegebene einzelne Zeile aus, und die
nächste eingegebene Zeile ersetzt die ausgegebene Zeile in der Datei.

a - ANHÄNGEN (APPEND)

Dieser Befehl entspricht $+1i .

[#],[#],#,[#]c - EINEN ZEILENBEREICH KOPIEREN

Dieser Befehl kopiert einen Block aufeinanderfolgender Zeilen an eine andere
Stelle im Puffer. Die Parameter sind:

  * Die erste Zeile, die kopiert werden soll (Standard = aktuelle Zeile)
  * Die letzte Zeile, die kopiert werden soll (Standard = aktuelle Zeile)
  * Die Zeile, vor der der Zeilenblock eingefügt werden soll
  * Die Anzahl der Kopien des Blocks (Standard = 1)

Nachdem edlin die Zeilen kopiert hat, können Sie den Befehl l (list)
verwenden, um die korrekt neu nummerierten Zeilen zu sehen.

[#][,#]d - EINEN ZEILENBLOCK LÖSCHEN

Dieser Befehl löscht einen Block aufeinanderfolgender Zeilen. Die Parameter
sind die erste und letzte Zeile des zu löschenden Blocks.

Wenn Sie den zweiten Parameter weglassen, wird nur die im ersten Parameter
angegebene Zeile gelöscht. Wenn Sie den ersten Parameter weglassen, wird der
Block von der aktuellen Zeile bis zu der im zweiten Parameter angegebenen
Zeile gelöscht (achten Sie darauf, das Komma anzugeben). Das Weglassen beider
Parameter führt dazu, dass die aktuelle Zeile gelöscht wird.

e dateiname - SPEICHERN UND BEENDEN

Dieser Befehl entspricht

  w dateiname

gefolgt von

  q

[#]i - EINFÜGEMODUS

Dieser Befehl startet den Einfügemodus. Sein Parameter ist die Zeilennummer,
vor der Zeilen eingefügt werden sollen.

Während des Einfügens ist die Eingabe von Escape-Sequenzen wie den oben
genannten zulässig. Um den Einfügemodus zu beenden, geben Sie einen Punkt (.)
in einer ansonsten leeren Zeile ein (wenn Sie eine Zeile benötigen, die nur
einen Punkt enthält, maskieren Sie diesen mit einem Backslash).

Nach dem Beenden des Einfügemodus wird die Zeile nach dem eingefügten Text
zur aktuellen Zeile, es sei denn, die Einfügung wurde am Ende des Textes
angehängt; in diesem Fall wird die letzte Zeile des Textes zur neuen
aktuellen Zeile.

[#][,#]l - ZEILEN AUFLISTEN

Dieser Befehl listet Textzeilen auf dem Bildschirm auf. Wenn Sie den ersten
Parameter weglassen, beginnt der Befehl mit der Anzeige von Text 11 Zeilen
vor der aktuellen Zeile. Das Weglassen des zweiten Parameters zeigt eine
Seite Text beginnend mit der im ersten Parameter angegebenen Zeile. Das
Weglassen beider Parameter zeigt eine Seite Text beginnend 11 Zeilen vor der
aktuellen Zeile. In allen Fällen gilt: Wenn die Anzahl der Zeilen größer ist
als die Anzahl der Zeilen auf einer Seite, fordert das Programm nach jeder
Seite zur Fortsetzung auf.

[#],[#],#m - BLOCK VERSCHIEBEN

Dieser Befehl verschiebt einen Textblock vor die im dritten Parameter
angegebene Zeilennummer. Dies entspricht dem Kopieren und anschließenden
Löschen des ursprünglichen Blocks.

[#][,#]p - SEITE (PAGE)

Dies ähnelt dem Befehl l, außer dass die Standardseite mit der aktuellen
Zeile beginnt anstatt 11 Zeilen davor.

q - BEENDEN (QUIT)

Selbsterklärend.

[#][,#][?]r$,$ - ZEICHENKETTE ERSETZEN

Dieser Befehl ersetzt alle Vorkommen der ersten Zeichenkette durch die zweite
zwischen den zwei angegebenen Zeilen. Ein weggelassener erster Parameter
startet das Ersetzen in der Zeile nach der aktuellen Zeile; ein
weggelassener zweiter Parameter stoppt das Ersetzen an der letzten Zeile der
Datei.

Das Fragezeichen bewirkt eine Bestätigungsmeldung vor dem Ersetzen.

[#][,#][?]s$ - NACH TEILSTRING SUCHEN

Dieser Befehl sucht nach der ersten Zeile, die den angegebenen Teilstring
enthält. Das Weglassen des ersten Parameters bewirkt, dass die Suche in der
aktuellen Zeile beginnt; das Weglassen des zweiten bewirkt, dass die Suche
am Ende der Datei stoppt.

Das Fragezeichen bewirkt, dass die Suche eine Bestätigungsmeldung sendet.
Wird die Bestätigung mit "N" (Nein) beantwortet, wird die Suche
fortgesetzt; ein "J" (Ja) beendet sie.

Die aktuelle Zeile wird auf die Zeile zurückgesetzt, an der die Suche
erfolgreich endete.

[#]t dateiname - DATEI ÜBERTRAGEN

Dieser Befehl bewirkt, dass der Inhalt der Datei vor der angegebenen
Zeilennummer eingefügt wird. Wenn die Zeilennummer weggelassen wird, ist der
Standardwert die aktuelle Zeile.

[#]w dateiname - DATEI SCHREIBEN

Dieser Befehl bewirkt, dass die erste durch den Parameter angegebene Anzahl
von Zeilen in die angegebene Datei geschrieben wird. Wenn der Parameter
weggelassen wird, werden alle Zeilen im Puffer in die Datei geschrieben.

EDLIN 2.14

Le programme edlin est un petit éditeur de lignes, écrit pour FreeDOS comme
un clone fonctionnel de l'ancien programme MS-DOS edlin. Il diffère de la
version MS sur deux points : d'abord, c'est un logiciel libre, et ensuite,
l'interface utilisateur est légèrement différente par endroits. La raison
de cette différence est d'éviter à l'utilisateur d'avoir à taper des
caractères de contrôle imposés par la syntaxe de MS-edlin.

APPEL DE EDLIN

Le programme edlin est appelé soit seul :

  edlin

soit avec un nom de fichier qui est lu initialement :

  edlin fichier

COMMANDES INTERNES D'EDLIN

Dans les listes de commandes suivantes, les crochets représentent des
constructions optionnelles. Le symbole # représente un numéro de ligne,
qui est soit une suite de chiffres, un point (.) représentant la ligne
courante, un signe dollar ($) représentant la dernière ligne du fichier,
ou un numéro de ligne ajouté ou soustrait d'un autre numéro de ligne, de
sorte que :

  .,.+1p

affiche la ligne courante et la ligne suivante.

Un signe dollar dans les listes suivantes représente une chaîne de
caractères. Ils peuvent être entourés de guillemets simples ou doubles et
peuvent contenir les séquences d'échappement suivantes :

  \a - alerte (bip)
  \b - retour arrière
  \e - échappement
  \f - saut de page
  \t - tabulation horizontale
  \v - tabulation verticale
  \" - guillemet double
  \' - guillemet simple
  \. - point
  \\ - barre oblique inverse
  \xXX où X est un chiffre hexadécimal (0-9, a-f) - constante hexadécimale
  \dddd où d est un chiffre décimal (0-9) - constante décimale
  \OOO où O est un chiffre octal (0-7) - constante octale
  \^C - caractère de contrôle

Le programme edlin supporte la syntaxe suivante :

# - MODIFIER UNE SEULE LIGNE

edlin affiche la ligne indiquée par le numéro, et la ligne suivante saisie
remplace la ligne affichée dans le fichier.

a - AJOUTER (APPEND)

Cette commande est équivalente à $+1i .

[#],[#],#,[#]c - COPIER UNE PLAGE DE LIGNES

Cette commande copie un bloc de lignes consécutives vers un autre
emplacement du tampon. Les paramètres sont :

  * La première ligne à copier (par défaut = ligne courante)
  * La dernière ligne à copier (par défaut = ligne courante)
  * La ligne avant laquelle insérer le bloc de lignes
  * Le nombre de copies du bloc (par défaut = 1)

Après la copie, vous pouvez utiliser la commande l (liste) pour voir les
lignes renumérotées.

[#][,#]d - SUPPRIMER UN BLOC DE LIGNES

Cette commande supprime un bloc de lignes consécutives. Les paramètres sont
la première et la dernière ligne du bloc à supprimer.

Si vous omettez le second paramètre, seule la ligne spécifiée au premier
est supprimée. Si vous omettez le premier, le bloc de la ligne courante
jusqu'à la ligne spécifiée au second est supprimé (veillez à inclure la
virgule). L'omission des deux paramètres supprime la ligne courante.

e fichier - ENREGISTRER ET QUITTER

Cette commande est équivalente à :

  w fichier
  q

[#]i - MODE INSERTION

Cette commande active le mode insertion. Son paramètre est le numéro de
ligne avant lequel vous souhaitez insérer du texte.

Pendant l'insertion, les séquences d'échappement ci-dessus sont valides.
Pour quitter, tapez un point (.) sur une ligne vide (si vous avez besoin
d'une ligne avec juste un point, utilisez un caractère d'échappement).

Après avoir quitté, la ligne après le texte inséré devient la ligne
courante, sauf si l'insertion a été ajoutée à la fin, auquel cas la
dernière ligne devient la nouvelle ligne courante.

[#][,#]l - LISTER LES LIGNES

Affiche les lignes à l'écran. Sans premier paramètre, l'affichage commence
11 lignes avant la ligne courante. Sans second paramètre, une page est
affichée. Sans les deux, une page commence 11 lignes avant la ligne
courante. Si le nombre de lignes dépasse une page, le programme demande
confirmation après chaque page.

[#],[#],#m - DÉPLACER UN BLOC

Déplace un bloc de texte avant la ligne spécifiée au troisième paramètre.
C'est équivalent à copier, puis supprimer le bloc original.

[#][,#]p - PAGE

Semblable à l (liste), sauf que la page par défaut commence à la ligne
courante au lieu de 11 lignes avant.

q - QUITTER

Quitte sans enregistrer les modifications supplémentaires.

[#][,#][?]r$,$ - REMPLACER UNE CHAÎNE

Remplace toutes les occurrences de la première chaîne par la seconde entre
les deux lignes spécifiées. Sans premier paramètre, le remplacement débute
après la ligne courante ; sans second, il s'arrête à la fin du fichier.

Le point d'interrogation (?) demande confirmation avant chaque remplacement.

[#][,#][?]s$ - RECHERCHER UNE SOUS-CHAÎNE

Recherche la première ligne contenant la sous-chaîne. Sans premier
paramètre, la recherche commence à la ligne courante ; sans second, elle
finit à la fin du fichier.

Le point d'interrogation (?) active une confirmation. Répondre "N" (Non)
continue la recherche ; répondre "O" (Oui) l'arrête. En cas de succès, la
ligne courante est fixée sur la ligne trouvée.

[#]t fichier - TRANSFÉRER UN FICHIER

Insère le contenu d'un fichier externe avant la ligne spécifiée. Par
défaut, il s'agit de la ligne courante.

[#]w fichier - ÉCRIRE LE FICHIER

Écrit le nombre spécifié de lignes dans le fichier. Sans paramètre, tout
le tampon est écrit.

EDLIN 2.14

El programa edlin es un pequeño editor de líneas, escrito para FreeDOS como
un clon funcional del antiguo programa edlin de MS-DOS. Se diferencia del
MS-edlin en que, primero, es software libre y, segundo, la interfaz de
usuario es ligeramente distinta en algunos puntos. El motivo del cambio es
evitar que el usuario tenga que teclear caracteres de control exigidos por
la sintaxis original de MS-edlin.

INVOCACIÓN DE EDLIN

El programa edlin se invoca solo:

  edlin

o con un nombre de archivo que se lee al inicio:

  edlin archivo

COMANDOS INTERNOS DE EDLIN

En las siguientes listas, los corchetes representan elementos opcionales.
El símbolo # representa un número de línea, que puede ser una serie de
dígitos, un punto (.) para la línea actual, un signo de dólar ($) para la
última línea del archivo, o un número sumado o restado a otro, de modo que:

  .,.+1p

muestra la línea actual y la siguiente.

Un signo de dólar en las listas representa una cadena de caracteres. Pueden
ir entre comillas simples o dobles y contener estas secuencias de escape:

  \a - alerta (pitido)
  \b - retroceso
  \e - escape
  \f - salto de página
  \t - tabulación horizontal
  \v - tabulación vertical
  \" - comilla doble
  \' - comilla simple
  \. - punto
  \\ - barra invertida
  \xXX (X es dígito hexa 0-9, a-f) - constante hexadecimal
  \dddd (d es dígito decimal 0-9) - constante decimal
  \OOO (O es dígito octal 0-7) - constante octal
  \^C - carácter de control

El programa edlin admite la siguiente sintaxis:

# - EDITAR UNA SOLA LÍNEA

edlin muestra la línea indicada por el número, y la siguiente línea que se
introduzca reemplazará a dicha línea en el archivo.

a - AÑADIR (APPEND)

Este comando equivale a $+1i .

[#],[#],#,[#]c - COPIAR UN RANGO DE LÍNEAS

Copia un bloque de líneas consecutivas a otra ubicación del búfer. Los
parámetros son:

  * Primera línea a copiar (por defecto = línea actual)
  * Última línea a copiar (por defecto = línea actual)
  * Línea antes de la cual insertar el bloque
  * Número de veces a copiar el bloque (por defecto = 1)

Tras copiar, puede usar l (list) para ver la nueva numeración.

[#][,#]d - ELIMINAR UN BLOQUE DE LÍNEAS

Elimina un bloque de líneas consecutivas. Los parámetros definen la primera
y última línea del bloque.

Si omite el segundo parámetro, solo borra la línea del primero. Si omite el
primero, borra desde la línea actual hasta la del segundo (incluya la
coma). Si omite ambos, se borra la línea actual.

e archivo - GUARDAR Y SALIR

Equivale a:

  w archivo
  q

[#]i - MODO INSERCIÓN

Entra en modo inserción. El parámetro es la línea antes de la cual se
insertará el texto.

Al insertar, se permiten secuencias de escape. Para salir, escriba un punto
(.) en una línea vacía (si necesita una línea con solo un punto, use un
carácter de escape).

Al salir, la línea tras el texto insertado pasa a ser la actual, salvo si
se añadió al final, en cuyo caso la última línea será la nueva actual.

[#][,#]l - LISTAR LÍNEAS

Muestra líneas en pantalla. Sin primer parámetro, empieza 11 líneas antes
de la actual. Sin segundo parámetro, muestra una página. Sin ambos, muestra
una página desde 11 líneas antes de la actual. Si hay más líneas que una
página, el programa pausará tras cada pantalla.

[#],[#],#m - MOVER BLOQUE

Mueve un bloque de texto antes de la línea indicada en el tercer parámetro.
Es como copiar y luego borrar el original.

[#][,#]p - PÁGINA

Similar a l, pero la página empieza por defecto en la línea actual en vez
de 11 líneas antes.

q - SALIR (QUIT)

Sale sin guardar más cambios.

[#][,#][?]r$,$ - REEMPLAZAR CADENA

Reemplaza la primera cadena por la segunda entre las líneas indicadas. Sin
primer parámetro, empieza tras la línea actual; sin segundo, para al final
del archivo.

El signo de interrogación (?) pide confirmación antes de cada reemplazo.

[#][,#][?]s$ - BUSCAR CADENA

Busca la primera línea con la cadena indicada. Sin primer parámetro, busca
desde la actual; sin segundo, busca hasta el final.

El signo de interrogación (?) activa la confirmación. Responder "N" (No)
sigue buscando; "S" (Sí) detiene la búsqueda. Si tiene éxito, la línea
actual se sitúa donde terminó la búsqueda.

[#]t archivo - TRANSFERIR ARCHIVO

Inserta el contenido de un archivo externo antes de la línea indicada. Por
defecto es la línea actual.

[#]w archivo - ESCRIBIR ARCHIVO

Escribe el número de líneas indicado en el archivo. Sin parámetro, escribe
todo el búfer.

EDLIN 2.14

Il programma edlin è un piccolo editor di righe, scritto per FreeDOS come
clone funzionale del vecchio programma MS-DOS edlin. Si differenzia dalla
versione MS perché, in primo luogo, è software libero e, in secondo luogo,
l'interfaccia utente è leggermente diversa in alcuni punti. Il motivo della
differenza è evitare che l'utente debba digitare caratteri di controllo
richiesti dalla sintassi di MS-edlin.

AVVIO DI EDLIN

Il programma edlin viene avviato da solo:

  edlin

o con un nome di file che viene caricato inizialmente:

  edlin file

COMANDI INTERNI DI EDLIN

Nelle seguenti liste di comandi, le parentesi quadre rappresentano elementi
opzionali. Il simbolo # rappresenta un numero di riga, che può essere una
stringa di cifre, un punto (.) per la riga corrente, un segno di dollaro ($)
per l'ultima riga del file, o un numero aggiunto o sottratto a un altro,
cosicché:

  .,.+1p

stampa la riga corrente e la riga successiva.

Un segno di dollaro nelle liste rappresenta una stringa di caratteri. Esse
possono essere racchiuse tra virgolette singole o doppie e contenere le
seguenti sequenze di escape:

  \a - segnale acustico (alert)
  \b - backspace
  \e - escape
  \f - formfeed
  \t - tabulazione orizzontale
  \v - tabulazione verticale
  \" - virgolette doppie
  \' - virgolette singole
  \. - punto
  \\ - backslash
  \xXX (X è cifra esadecimale) - costante esadecimale
  \dddd (d è cifra decimale) - costante decimale
  \OOO (O è cifra ottale) - costante ottale
  \^C - carattere di controllo

Il programma edlin supporta la seguente sintassi:

# - MODIFICA UNA SINGOLA RIGA

edlin visualizza la riga indicata dal numero, e la riga successiva inserita
sostituisce quella visualizzata nel file.

a - AGGIUNGI (APPEND)

Questo comando è equivalente a $+1i .

[#],[#],#,[#]c - COPIA UN INTERVALLO DI RIGHE

Copia un blocco di righe consecutive in un'altra posizione del buffer. I
parametri sono:

  * La prima riga da copiare (predefinito = riga corrente)
  * L'ultima riga da copiare (predefinito = riga corrente)
  * La riga prima della quale inserire il blocco
  * Il numero di volte che si desidera copiare il blocco (predefinito = 1)

Dopo la copia, è possibile usare il comando l (list) per vedere le righe
rinumerate.

[#][,#]d - ELIMINA UN BLOCCO DI RIGHE

Elimina un blocco di righe consecutive. I parametri sono la prima e l'ultima
riga del blocco da eliminare.

Se si omette il secondo parametro, elimina solo la riga del primo. Se si
omette il primo, elimina dalla riga corrente a quella del secondo (includere
la virgola). Se si omettono entrambi, viene eliminata la riga corrente.

e nomefile - SALVA ED ESCI

Equivale a:

  w nomefile
  q

[#]i - MODALITÀ INSERIMENTO

Attiva la modalità inserimento. Il parametro è il numero di riga prima della
quale inserire il testo.

Durante l'inserimento, le sequenze di escape sono valide. Per uscire,
digitare un punto (.) su una riga vuota (per una riga con solo un punto,
usare un carattere di escape).

Dopo l'uscita, la riga successiva al testo inserito diventa la corrente, a
meno che l'inserimento non sia avvenuto alla fine, nel qual caso l'ultima
riga diventa la nuova riga corrente.

[#][,#]l - ELENCA RIGHE

Visualizza le righe a schermo. Senza primo parametro, inizia 11 righe prima
della corrente. Senza secondo, mostra una pagina. Senza entrambi, mostra
una pagina da 11 righe prima della corrente. Se le righe superano una
pagina, il programma chiederà conferma dopo ogni schermata.

[#],[#],#m - SPOSTA BLOCCO

Sposta un blocco di testo prima della riga indicata nel terzo parametro. È
come copiare e poi eliminare il blocco originale.

[#][,#]p - PAGINA

Simile a l, ma la pagina predefinita inizia dalla riga corrente invece di
11 righe prima.

q - ESCI (QUIT)

Esce senza salvare ulteriori modifiche.

[#][,#][?]r$,$ - SOSTITUISCI STRINGA

Sostituisce la prima stringa con la seconda tra le righe indicate. Senza
primo parametro, inizia dopo la riga corrente; senza secondo, termina
all'ultima riga del file.

Il punto interrogativo (?) chiede conferma prima di ogni sostituzione.

[#][,#][?]s$ - CERCA SOTTOSTRINGA

Cerca la prima riga contenente la stringa indicata. Senza primo parametro,
cerca dalla riga corrente; senza secondo, cerca fino alla fine del file.

Il punto interrogativo (?) attiva la conferma. Rispondere "N" (No) continua
la ricerca; "S" (Sì) la interrompe. Se riuscita, la riga corrente viene
impostata dove è terminata la ricerca.

[#]t nomefile - TRASFERISCI FILE

Inserisce il contenuto di un file esterno prima della riga indicata. Se
omessa, viene usata la riga corrente.

[#]w nomefile - SCRIVI FILE

Scrive il numero di righe indicato nel file. Senza parametro, scrive tutto
il buffer nel file.

O programa edlin é um pequeno editor de linhas, escrito para o FreeDOS como
um clone funcional do antigo programa edlin do MS-DOS. Difere do MS-edlin
porque, primeiro, é software livre e, segundo, a interface do utilizador é
ligeiramente diferente em alguns pontos. O motivo da diferença é evitar que
o utilizador tenha de digitar caracteres de controlo exigidos pela sintaxe
do MS-edlin.

INVOCAÇÃO DO EDLIN

O programa edlin é invocado ou sozinho:

  edlin

ou com um nome de arquivo que é lido inicialmente:

  edlin arquivo

COMANDOS INTERNOS DO EDLIN

Nas seguintes listas de comandos, os parênteses representam elementos
opcionais. O símbolo # representa um número de linha, que pode ser uma
sequência de dígitos, um ponto (.) representando a linha atual, um cifrão
($) representando a última linha do arquivo, ou um número somado ou
subtraído de outro, de modo que:

  .,.+1p

imprime a linha atual e a linha seguinte.

Um cifrão nas listas seguintes representa uma cadeia de caracteres. Podem
estar entre aspas simples ou duplas e conter as seguintes sequências de
escape:

  \a - alerta (bipe)
  \b - backspace
  \e - escape
  \f - alimentação de formulário
  \t - tabulação horizontal
  \v - tabulação vertical
  \" - aspas duplas
  \' - aspa simples
  \. - ponto
  \\ - barra invertida
  \xXX (X é dígito hexa 0-9, a-f) - constante hexadecimal
  \dddd (d é dígito decimal 0-9) - constante decimal
  \OOO (O é dígito octal 0-7) - constante octal
  \^C - caractere de controlo

O programa edlin suporta a seguinte sintaxe:

# - EDITAR UMA LINHA ÚNICA

O edlin exibe a linha indicada pelo número, e a linha seguinte introduzida
substitui a linha exibida no arquivo.

a - ANEXAR (APPEND)

Este comando é equivalente a $+1i .

[#],[#],#,[#]c - COPIAR UM INTERVALO DE LINHAS

Copia um bloco de linhas consecutivas para outro local no buffer. Os
parâmetros são:

  * A primeira linha a copiar (padrão = linha atual)
  * A última linha a copiar (padrão = linha atual)
  * A linha antes da qual inserir o bloco de linhas
  * O número de vezes que deseja copiar o bloco (padrão = 1)

Após copiar, pode usar o comando l (list) para ver as linhas renumeradas.

[#][,#]d - EXCLUIR UM BLOCO DE LINHAS

Exclui um bloco de linhas consecutivas. Os parâmetros são a primeira e a
última linha do bloco a ser excluído.

Se omitir o segundo parâmetro, exclui apenas a linha especificada no
primeiro. Se omitir o primeiro, exclui desde a linha atual até à linha do
segundo parâmetro (inclua a vírgula). Omitir ambos exclui a linha atual.

e nomearquivo - SALVAR E SAIR

Equivale a:

  w nomearquivo
  q

[#]i - MODO DE INSERÇÃO

Entra no modo de inserção. O parâmetro é o número da linha antes da qual
deseja inserir texto.

Ao inserir, as sequências de escape são válidas. Para sair, digite um ponto
(.) numa linha vazia (se precisar de uma linha apenas com um ponto, use um
caractere de escape).

Após sair, a linha depois do texto inserido torna-se a linha atual, exceto
se a inserção foi anexada ao fim; nesse caso, a última linha torna-se a
nova linha atual.

[#][,#]l - LISTAR LINHAS

Exibe linhas de texto no ecrã. Sem o primeiro parâmetro, começa 11 linhas
antes da atual. Sem o segundo, mostra uma página. Sem ambos, mostra uma
página desde 11 linhas antes da atual. Se o número de linhas for superior a
uma página, o programa fará uma pausa após cada ecrã.

[#],[#],#m - MOVER BLOCO

Move um bloco de texto para antes da linha indicada no terceiro parâmetro.
É como copiar e depois excluir o bloco original.

[#][,#]p - PÁGINA

Semelhante a l, mas a página padrão começa na linha atual em vez de 11
linhas antes.

q - SAIR (QUIT)

Sai sem salvar mais alterações.

[#][,#][?]r$,$ - SUBSTITUIR CADEIA

Substitui a primeira cadeia pela segunda entre as linhas indicadas. Sem o
primeiro parâmetro, começa após a linha atual; sem o segundo, para na
última linha do arquivo.

O ponto de interrogação (?) pede confirmação antes de cada substituição.

[#][,#][?]s$ - PROCURAR CADEIA

Procura a primeira linha que contém a cadeia indicada. Sem o primeiro
parâmetro, procura a partir da atual; sem o segundo, procura até ao fim.

O ponto de interrogação (?) ativa a confirmação. Responder "N" (Não)
continua a busca; "S" (Sim) encerra-a. Se for bem-sucedido, a linha atual
será definida onde a busca terminou.

[#]t nomearquivo - TRANSFERIR ARQUIVO

Insere o conteúdo de um arquivo externo antes da linha indicada. Por
padrão, é a linha atual.

[#]w nomearquivo - GRAVAR ARQUIVO

Grava o número de linhas indicado no arquivo. Sem parâmetro, grava todo o
buffer.

EDLIN 2.14

Het programma edlin is een compacte regeleditor, geschreven voor FreeDOS als
een functionele kloon van het oude MS-DOS-programma edlin. Het verschilt van
MS-edlin doordat het ten eerste vrije software is en ten tweede de
gebruikersinterface op enkele punten is aangepast. De reden hiervoor is om de
gebruiker de invoer van complexe besturingstekens te besparen die de syntax
van MS-edlin voorschreef.

EDLIN STARTEN

Het programma edlin wordt ofwel direct aangeroepen:

  edlin

of met een bestandsnaam die bij aanvang wordt ingelezen:

  edlin bestand

INTERNE COMMANDO'S VAN EDLIN

In de volgende commandolijsten staan vierkante haken voor optionele delen.
Het symbool # staat voor een regelnummer, wat een reeks cijfers kan zijn, een
punt (.) voor de huidige regel, een dollarteken ($) voor de laatste regel van
het bestand, of een regelnummer vermeerderd of verminderd met een ander
getal, zodat:

  .,.+1p

de huidige regel en de daaropvolgende regel weergeeft.

Een dollarteken ($) in de lijsten staat voor een tekenreeks (string). Deze
kunnen tussen enkele of dubbele aanhalingstekens staan en de volgende
escape-sequenties bevatten:

  \a - signaal (alert)
  \b - backspace
  \e - escape
  \f - formfeed
  \t - horizontale tab
  \v - verticale tab
  \" - dubbel aanhalingsteken
  \' - enkel aanhalingsteken
  \. - punt
  \\ - backslash
  \xXX (X is hex-cijfer 0-9, a-f) - hexadecimale constante
  \dddd (d is decimaal cijfer 0-9) - decimale constante
  \OOO (O is octaal cijfer 0-7) - octale constante
  \^C - besturingsteken (control character)

Het programma edlin ondersteunt de volgende syntax:

# - EEN ENKELE REGEL BEWERKEN

edlin toont de regel die door het nummer wordt aangegeven. De volgende
invoer vervangt deze regel volledig in het bestand.

a - TOEVOEGEN (APPEND)

Dit commando is gelijk aan de invoer van $+1i .

[#],[#],#,[#]c - REGELBEREIK KOPIËREN

Kopieert een blok opeenvolgende regels naar een andere positie in de buffer.
De parameters zijn:

  * Eerste regel die moet worden gekopieerd (standaard = huidige regel)
  * Laatste regel die moet worden gekopieerd (standaard = huidige regel)
  * Regel waavóór het blok moet worden ingevoegd
  * Aantal keren dat het blok moet worden gekopieerd (standaard = 1)

Na het kopiëren kunt u het l (list) commando gebruiken om de nieuwe
regelnummering te controleren.

[#][,#]d - REGELBLOK VERWIJDEREN

Verwijdert een blok opeenvolgende regels. De parameters bepalen de eerste en
laatste regel van het te verwijderen bereik.

Als u de tweede parameter weglaat, wordt alleen de regel uit de eerste
parameter verwijderd. Als u de eerste weglaat, wordt vanaf de huidige regel
tot aan de opgegeven regel verwijderd (gebruik de komma). Het weglaten van
beide parameters verwijdert de huidige regel.

e bestandsnaam - OPSLAAN EN STOPPEN

Dit is gelijk aan de reeks:

  w bestandsnaam
  q

[#]i - INVOEGMODUS (INSERT)

Activeert de invoegmodus. De parameter is het regelnummer waavóór de tekst
moet worden ingevoegd.

Tijdens het invoegen zijn escape-sequenties toegestaan. Om de modus te
verlaten, typt u een punt (.) op een overigens lege regel (als u een regel
met alleen een punt nodig heeft, gebruik dan een escape-teken).

Na het afsluiten wordt de regel na de ingevoegde tekst de huidige regel,
tenzij de tekst aan het einde werd toegevoegd; dan wordt de laatste regel de
nieuwe huidige regel.

[#][,#]l - REGELS WEERGEVEN (LIST)

Toont tekstregels op het scherm. Zonder eerste parameter begint de weergave
11 regels vóór de huidige regel. Zonder tweede parameter wordt een pagina
getoond vanaf de startregel. Zonder beide wordt een pagina getoond vanaf 11
regels vóór de huidige. Als er meer regels zijn dan op een scherm passen,
pauzeert het programma na elke pagina.

[#],[#],#m - BLOK VERPLAATSEN (MOVE)

Verplaatst een tekstblok naar de positie vóór de regel uit de derde
parameter. Dit werkt als kopiëren gevolgd door het wissen van het origineel.

[#][,#]p - PAGINA (PAGE)

Vergelijkbaar met l, maar de standaardweergave begint direct bij de huidige
regel in plaats van 11 regels daarvóór.

q - STOPPEN (QUIT)

Sluit het programma af zonder verdere wijzigingen op te slaan.

[#][,#][?]r$,$ - TEKENREEKS VERVANGEN (REPLACE)

Vervangt alle voorkomens van de eerste reeks door de tweede binnen het
opgegeven bereik. Zonder eerste parameter begint het proces na de huidige
regel; zonder tweede eindigt het bij de laatste regel van het bestand.

Het vraagteken (?) vraagt om bevestiging vóór elke vervanging.

[#][,#][?]s$ - ZOEKEN NAAR REEKS (SEARCH)

Zoekt naar de eerste regel die de opgegeven reeks bevat. Zonder eerste
parameter wordt vanaf de huidige regel gezocht, zonder tweede tot het eind.

Het vraagteken (?) activeert bevestiging. Bij "N" (Nee) gaat het zoeken
door, bij "J" (Ja) stopt het. Bij succes wordt de huidige regel ingesteld op
de gevonden locatie.

[#]t bestandsnaam - BESTAND OVERDRAGEN (TRANSFER)

Voegt de inhoud van een extern bestand in vóór de opgegeven regel. Indien
niet opgegeven, wordt de huidige regel gebruikt.

[#]w bestandsnaam - BESTAND SCHRIJVEN (WRITE)

Schrijft het opgegeven aantal regels naar het bestand. Zonder parameter
wordt de gehele bufferinhoud opgeslagen.

Programmet edlin är en kompakt radeditor, skriven för FreeDOS som en
funktionell klon av det gamla MS-DOS-programmet edlin. Det skiljer sig från

STARTA EDLIN

Programmet edlin startas antingen direkt:

  edlin

eller med ett filnamn som läses in vid start:

  edlin fil

I följande kommandolistor representerar hakparenteser valfria delar.
Symbolen # står för ett radnummer, vilket kan vara en sifferföljd, en punkt
(.) för den aktuella raden, ett dollartecken ($) för den sista raden i filen,
eller ett radnummer plus eller minus ett annat radnummer, så att:

  .,.+1p

skriver ut den aktuella raden och den efterföljande raden.

Ett dollartecken ($) i listorna representerar en teckensträng. Dessa kan
omges av antingen enkla eller dubbla citattecken och innehålla följande
escape-sekvenser:

  \a - signal (alert)
  \b - backsteg (backspace)
  \e - escape
  \f - sidmatning (formfeed)
  \t - horisontell tabulator
  \v - vertikal tabulator
  \" - dubbelt citattecken
  \' - enkelt citattecken
  \. - punkt
  \\ - bakåtstreck (backslash)
  \xXX (där X är hex-siffra 0-9, a-f) - hexadecimal konstant
  \dddd (där d är decimal siffra 0-9) - decimal konstant
  \OOO (där O är oktal siffra 0-7) - oktal konstant
  \^C - styrtecken (control character)

Programmet edlin stöder följande syntax:

# - REDIGERA EN ENSKILD RAD

edlin visar den rad som anges av numret, och nästa rad som skrivs in
ersätter den visade raden i filen.

a - LÄGG TILL (APPEND)

Detta kommando motsvarar att skriva $+1i .

[#],[#],#,[#]c - KOPIERA RADINTERVALL

Kopierar ett block av på varandra följande rader till en annan plats i
bufferten. Parametrarna är:

  * Första raden som ska kopieras (standard = aktuell rad)
  * Sista raden som ska kopieras (standard = aktuell rad)
  * Raden före vilken blocket ska infogas
  * Antal gånger blocket ska kopieras (standard = 1)

Efter kopieringen kan du använda kommandot l (list) för att se de korrekt
omnumrerade raderna.

[#][,#]d - TA BORT RADBLOCK

Tar bort ett block av på varandra följande rader. Parametrarna anger första
och sista raden i blocket som ska tas bort.

Om du utelämnar den andra parametern tas endast raden i den första
parametern bort. Om du utelämnar den första tas blocket från aktuell rad
till raden i den andra parametern bort (se till att inkludera kommat). Om
båda parametrarna utelämnas tas den aktuella raden bort.

e filnamn - SPARA OCH AVSLUTA

Detta motsvarar kommandoföljden:

  w filnamn
  q

[#]i - INFOGNINGSLÄGE (INSERT)

Aktiverar infogningsläget. Parametern är radnumret före vilket text ska
infogas.

Vid infogning är escape-sekvenser tillåtna. För att avsluta läget skriver du
en punkt (.) på en i övrigt tom rad (om du behöver en rad med bara en punkt,
använd escape-tecken).

Efter avslutat läge blir raden efter den infogade texten den aktuella raden,
såvida inte texten lades till i slutet; då blir den sista raden den nya
aktuella raden.

[#][,#]l - LISTA RADER

Visar textrader på skärmen. Utan första parameter börjar visningen 11 rader
före den aktuella raden. Utan andra parameter visas en sida text. Utan båda
visas en sida text från 11 rader före aktuell rad. Om antalet rader är fler
än vad som ryms på en skärm kommer programmet att pausa efter varje sida.

[#],[#],#m - FLYTTA BLOCK (MOVE)

Flyttar ett textblock till positionen före radnumret i den tredje
parametern. Det fungerar som en kopiering följt av borttagning av originalet.

[#][,#]p - SIDA (PAGE)

Liknar l, men standardvisningen börjar vid den aktuella raden istället för
11 rader före.

q - AVSLUTA (QUIT)

Avslutar programmet utan att spara ytterligare ändringar.

[#][,#][?]r$,$ - ERSÄTT STRÄNG (REPLACE)

Ersätter alla förekomster av den första strängen med den andra inom det
valda intervallet. Utan första parameter börjar ersättningen efter aktuell
rad; utan andra slutar den vid filens sista rad.

Frågetecknet (?) begär bekräftelse före varje ersättning.

[#][,#][?]s$ - SÖK EFTER STRÄNG (SEARCH)

Söker efter den första raden som innehåller den angivna strängen. Utan
första parameter söks från aktuell rad, utan andra till slutet av filen.

Frågetecknet (?) aktiverar bekräftelse. Vid "N" (Nej) fortsätter sökningen,
vid "J" (Ja) avslutas den. Vid träff sätts aktuell rad till den plats där
sökningen avslutades.

[#]t filnamn - ÖVERFÖR FIL (TRANSFER)

Infogar innehållet i en extern fil före det angivna radnumret. Om inget
anges används aktuell rad.

[#]w filnamn - SKRIV FIL (WRITE)

Skriver det angivna antalet rader till filen. Utan parameter skrivs hela
buffertens innehåll till filen.

EDLIN IEKŠĒJĀS KOMANDAS

Turpmākajos komandu sarakstos kvadrātiekavas apzīmē izvēles elementus.
Simbols # apzīmē rindas numuru, kas var būt ciparu virkne, punkts (.), kas

  \f - lapas padeve (formfeed)
  \t - horizontālā tabulācija

  \" - dubultpēdiņas

  \dddd (kur d ir decimālais cipars 0-9) - decimālā konstante

  \^C - vadības rakstzīme (control character)

EDLIN 2.14

Программа edlin — это компактный строчный редактор, написанный для FreeDOS
как функциональный клон старой программы edlin из MS-DOS. Он отличается от
версии MS edlin тем, что, во-первых, является свободным программным
обеспечением, и, во-вторых, пользовательский интерфейс в нескольких местах
немного оптимизирован. Причина изменений в том, чтобы избавить пользователя
от необходимости вводить сложные управляющие символы, которых требовал
синтаксис MS-edlin.

ЗАПУСК EDLIN

Программа edlin запускается либо сама по себе:

  edlin

либо с указанием имени файла, который загружается при старте:

  edlin файл

ВНУТРЕННИЕ КОМАНДЫ EDLIN

В следующих списках команд квадратные скобки обозначают необязательные
параметры. Символ # обозначает номер строки, который может быть набором
цифр, точкой (.) для обозначения текущей строки, знаком доллара ($) для
последней строки файла или номером строки, измененным путем сложения или
вычитания, так что:

  .,.+1p

выводит текущую и следующую за ней строки.

Знак доллара ($) в списках обозначает текстовую строку. Строки могут быть
заключены в одинарные или двойные кавычки и могут содержать следующие
управляющие последовательности (escape-последовательности):

  \a - звуковой сигнал (alert)
  \b - возврат на шаг (backspace)
  \e - escape
  \f - прогон страницы (formfeed)
  \t - горизонтальная табуляция
  \v - вертикальная табуляция
  \" - двойная кавычка
  \' - одинарная кавычка
  \. - точка
  \\ - обратная косая черта (backslash)
  \xXX (где X — hex-цифра 0-9, a-f) — шестнадцатеричная константа
  \dddd (где d — десятичная цифра 0-9) — десятичная константа
  \OOO (где O — восьмеричная цифра 0-7) — восьмеричная константа
  \^C — управляющий символ (Control)

Программа edlin поддерживает следующий синтаксис:

# - РЕДАКТИРОВАНИЕ ОДНОЙ СТРОКИ

edlin выводит строку с указанным номером, и следующая введенная строка
полностью заменяет её в файле.

a - ДОБАВИТЬ (APPEND)

Эта команда эквивалентна вводу $+1i .

[#],[#],#,[#]c - КОПИРОВАНИЕ ДИАПАЗОНА СТРОК

Копирует блок последовательных строк в другое место буфера. Параметры:

  * Первая строка для копирования (по умолчанию = текущая)
  * Последняя строка для копирования (по умолчанию = текущая)
  * Строка, перед которой нужно вставить блок
  * Количество копий блока (по умолчанию = 1)

После копирования можно использовать команду l (list), чтобы увидеть
новую нумерацию строк.

[#][,#]d - УДАЛЕНИЕ БЛОКА СТРОК

Удаляет блок последовательных строк. Параметры задают первую и последнюю
строки удаляемого диапазона.

Если второй параметр опущен, удаляется только строка из первого параметра.
Если опущен первый параметр, удаляется диапазон от текущей строки до
указанной (запятая обязательна). Если опущены оба параметра, удаляется
текущая строка.

e имя_файла - СОХРАНИТЬ И ВЫЙТИ

Эквивалентно последовательности команд:

  w имя_файла
  q

[#]i - РЕЖИМ ВСТАВКИ (INSERT)

Активирует режим вставки. Параметр задает номер строки, перед которой будет
вставлен новый текст.

При вставке можно использовать управляющие последовательности, указанные
выше. Чтобы выйти из режима, введите точку (.) в пустой строке (если нужна

После выхода строка, следующая за вставленным текстом, становится текущей,

Выводит строки текста на экран. Без первого параметра вывод начинается за
11 строк до текущей. Без второго параметра выводится страница текста. Без
обоих параметров выводится страница текста, начиная за 11 строк до текущей.
Если строк больше, чем умещается на экране, программа сделает паузу.

[#],[#],#m - ПЕРЕМЕСТИТЬ БЛОК (MOVE)

Перемещает блок текста перед строкой, указанной в третьем параметре. Это
действует как копирование с последующим удалением оригинала.

Аналогично l, но вывод по умолчанию начинается с текущей строки, а не за
11 строк до неё.

Завершает работу без дальнейшего сохранения.

[#][,#][?]r$,$ - ЗАМЕНА СТРОКИ (REPLACE)

Заменяет все вхождения первой строки на вторую в заданном диапазоне. Без
первого параметра замена начинается после текущей строки; без второго —
заканчивается на последней строке файла.

Вопросительный знак (?) вызывает запрос подтверждения перед каждой заменой.

Програма edlin — це невеликий рядковий редактор, написаний для FreeDOS як
функціональний клон старої програми edlin з MS-DOS. Він відрізняється від
версії MS edlin тим, що, по-перше, є вільним програмним забезпеченням, і,
по-друге, інтерфейс користувача в кількох місцях дещо оптимізовано. Причина
змін полягає в тому, щоб позбавити користувача необхідності вводити складні
керуючі символи, яких вимагав синтаксис MS-edlin.

Програма edlin запускається або сама по собі:

  edlin

або з вказанням імені файлу, який завантажується при старті:

  edlin файл

ВНУТРІШНІ КОМАНДИ EDLIN

У наступних списках команд квадратні дужки позначають необов'язкові
параметри. Символ # позначає номер рядка, який може бути послідовністю цифр,
крапкою (.) для позначення поточного рядка, знаком долара ($) для останнього
рядка файлу або номером рядка, зміненим шляхом додавання або віднімання,
так що:

  .,.+1p

виводить поточний та наступний за ним рядки.

Знак долара ($) у списках позначає текстовий рядок. Рядки можуть бути взяті
в одинарні або подвійні лапки та можуть містити наступні керуючі
послідовності (escape-послідовності):

  \a - звуковий сигнал (alert)
  \b - крок назад (backspace)
  \e - escape
  \f - прогін сторінки (formfeed)
  \t - горизонтальна табуляція
  \v - вертикальна табуляція
  \" - подвійні лапки
  \' - одинарні лапки
  \. - крапка
  \\ - зворотна коса риска (backslash)
  \xXX (де X — hex-цифра 0-9, a-f) — шістнадцяткова константа
  \dddd (де d — десяткова цифра 0-9) — десяткова константа
  \OOO (де O — вісімкова цифра 0-7) — вісімкова константа
  \^C — керуючий символ (Control)

Програма edlin підтримує наступний синтаксис:

# - РЕДАГУВАННЯ ОДНОГО РЯДКА

edlin виводить рядок із вказаним номером, і наступний введений рядок
повністю замінює його у файлі.

Ця команда еквівалентна введенню $+1i .

  * Перший рядок для копіювання (за замовчуванням = поточний)
  * Останній рядок для копіювання (за замовчуванням = поточний)
  * Рядок, перед яким потрібно вставити блок
  * Кількість копій блоку (за замовчуванням = 1)

Після копіювання можна використовувати команду l (list), щоб побачити
нову нумерацію рядків.

діапазону, що видаляється.

Якщо другий параметр пропущено, видаляється лише рядок з першого параметра.
Якщо пропущено перший параметр, видаляється діапазон від поточного рядка до
вказаного (кома обов'язкова). Якщо пропущено обидва параметри, видаляється
поточний рядок.

  w ім'я_файлу
  q

При вставці можна використовувати керуючі послідовності, вказані вище. Щоб

паузу.

Переміщує блок тексту перед рядком, вказаним у третьому параметрі. Це діє як
копіювання з подальшим видаленням оригіналу.

[#][,#]p - ПОСТОРІНКОВИЙ ВИВІД (PAGE)

Аналогічно l, але виведення за замовчуванням починається з поточного рядка,

Завершує роботу без подальшого збереження.

[#][,#][?]r$,$ - ЗАМІНИТИ РЯДОК (REPLACE)

Замінює всі входження першого рядка на другий у заданому діапазоні. Без
першого параметра заміна починається після поточного рядка; без другого —
закінчується на останньому рядку файлу.

Знак питання (?) викликає запит підтвердження перед кожною заміною.

[#][,#][?]s$ - ПОШУК ПІДРЯДКА (SEARCH)

Шукає перший рядок, що містить вказаний підрядок. Без першого параметра
пошук йде від поточного рядка, без другого — до кінця файлу.

Знак питання (?) активує підтвердження. При відповіді «Н» (Ні) пошук
продовжується, при «Т» (Так) — завершується. У разі успіху поточним стає
рядок, на якому пошук закінчився.

[#]t ім'я_файлу - ПЕРЕНЕСЕННЯ ФАЙЛУ (TRANSFER)

Вставляє вміст зовнішнього файлу перед вказаним рядком. За замовчуванням
використовується поточний рядок.

[#]w ім'я_файлу - ЗАПИСАТИ ФАЙЛ (WRITE)

Записує вказану кількість рядків у файл. Без параметра зберігається весь
вміст буфера.

Liigutab tekstiploki kolmandas parameetris määratud rea ette. See toimib nagu
kopeerimine koos algse ploki kustutamisega.

[#][,#]p - LEHEKÜLG (PAGE)

Sarnane käsuga l, kuid vaikimisi lehekülg algab praegusest reast (mitte 11
rida varem).

q - VÄLJU (QUIT)

Enesestmõistetav. Väljub ilma täiendava salvestamiseta.

[#][,#][?]r$,$ - SÕNE ASENDAMINE

Asendab kõik esimese sõne esinemised teisega valitud vahemikus. Ilma esimese
parameetrita algab asendamine pärast praegust rida; ilma teiseta lõpeb see
faili lõpus.

Küsimärk (?) küsib enne asendamist kinnitust.

[#][,#][?]s$ - ALAMSÕNE OTSIMINE

Otsib esimest rida, mis sisaldab määratud sõnet. Ilma esimese parameetrita
algab otsing praegusest reast, ilma teiseta kestab faili lõpuni.

Küsimärk (?) aktiveerib kinnituse. Vastus "E" (Ei) jätkab otsingut, vastus
"J" (Jah) lõpetab selle. Edu korral seatakse praeguseks reaks leitud asukoht.

[#]t failinimi - FAILI ÜLEKANDMINE

Lisab välise faili sisu määratud rea ette. Vaikimisi on selleks praegune rida.

[#]w failinimi - FAILI KIRJUTAMINE

Kirjutab määratud arvu ridu faili. Ilma parameetrita kirjutatakse kogu puhvri
sisu faili.

EDLIN 2.14

Programa edlin yra nedidelė eilučių redagavimo priemonė, sukurta FreeDOS
sistemai kaip senosios MS-DOS edlin programos funkcinis klonas. Ji skiriasi
nuo MS edlin tuo, kad, pirma, tai yra laisvoji programinė įranga, ir, antra,
vartotojo sąsaja keliose vietose yra šiek tiek kitokia. Šie skirtumai sukurti
tam, kad vartotojui nereikėtų įvedinėti valdymo simbolių, kurių reikalauja
MS edlin sintaksė.

EDLIN PALEIDIMAS

Programa edlin paleidžiamas arba viena:

  edlin

arba su failo vardu, kuris nuskaitomas paleidimo metu:

  edlin failas

EDLIN VIDINĖS KOMANDOS

Toliau pateiktuose komandų sąrašuose laužtiniai skliaustai nurodo pasirinktines
dalis. Simbolis # reiškia eilutės numerį, kuris gali būti skaitmenų eilutė,
taškas (.), nurodantis dabartinę eilutę, dolerio ženklas ($), nurodantis
paskutinę failo eilutę, arba eilutės numeris, prie kurio pridedamas arba
atimamas kitas eilutės numeris, pavyzdžiui:

  .,.+1p

išveda dabartinę eilutę ir po jos einančią eilutę.

Dolerio ženklas sąrašuose reiškia simbolių eilutę. Jos gali būti rašomos
viengubose arba dvigubose kabutėse ir gali turėti šias valdymo sekas
(escape sequences):

  \a - signalas (alert)
  \b - grįžimo klavišas (backspace)
  \e - escape
  \f - puslapio postūmis (formfeed)
  \t - horizontalus tabuliavimas
  \v - vertikalus tabuliavimas
  \" - dvigubos kabutės
  \' - vienguba kabutė
  \. - taškas
  \\ - atvirkštinis pasvirasis brūkšnys
  \xXX (kur X yra šešioliktainis skaitmuo 0-9, a-f) - šešioliktainė konstanta
  \dddd (kur d yra dešimtainis skaitmuo 0-9) - dešimtainė konstanta
  \OOO (kur O yra aštuntainis skaitmuo 0-7) - aštuntainė konstanta
  \^C - valdymo simbolis (control character)

Programa edlin palaiko šią sintaksę:

# - VIENOS EILUTĖS REDAGAVIMAS

edlin išveda numeriu nurodytą eilutę, o kita įvesta eilutė pakeičia išvestąją
eilutę faile.

a - PRIDĖTI (APPEND)

Ši komanda yra ekvivalenti $+1i .

[#],[#],#,[#]c - EILUČIŲ DIAPAZONO KOPIJAVIMAS

Ši komanda nukopijuoja gretimų eilučių bloką į kitą buferio vietą. Parametrai:

  * Pirmoji kopijuojama eilutė (numatytoji = dabartinė eilutė)
  * Paskutinė kopijuojama eilutė (numatytoji = dabartinė eilutė)
  * Eilutė, prieš kurią norite įterpti bloką
  * Bloko kopijavimo kartų skaičius (numatytasis = 1)

Nukopijavę eilutes, galite naudoti komandą l (list), kad pamatytumėte teisingai
pernumeruotas eilutes.

[#][,#]d - EILUČIŲ BLOKO IŠTRYNIMAS

Ši komanda ištrina gretimų eilučių bloką. Parametrai yra pirmoji ir paskutinė
trinamo bloko eilutės.

Jei praleisite antrąjį parametrą, bus ištrinta tik pirmajame nurodyta eilutė.
Jei praleisite pirmąjį, bus ištrintas blokas nuo dabartinės iki antrajame
parametre nurodytos eilutės (nepamirškite kablelio). Praleidus abu parametrus,
ištrinama dabartinė eilutė.

e failas - IŠSAUGOTI IR IŠEITI

Ši komanda yra ekvivalenti sekai:

  w failas
  q

[#]i - ĮTERPIMO REŽIMAS (INSERT)

Ši komanda įjungia įterpimo režimą. Parametras yra eilutės numeris, prieš
kurį norite įterpti naują tekstą.

Įterpimo metu leidžiama naudoti aukščiau nurodytas valdymo sekas. Norėdami
išeiti iš režimo, įrašykite tašką (.) tuščioje eilutėje (jei reikia eilutės
tik su tašku, naudokite valdymo simbolį).

Išėjus iš režimo, eilutė po įterpto teksto tampa dabartine eilute, nebent
tekstas buvo pridėtas gale; tokiu atveju paskutinė eilutė tampa dabartine.

[#][,#]l - IŠVARDINTI EILUTES (LIST)

Išveda teksto eilutes ekrane. Be pirmojo parametro rodymas pradedamas 11
eilučių prieš dabartinę. Be antrojo parametro rodomas puslapis teksto nuo
pradžios taško. Be abiejų parametrų rodomas puslapis nuo 11 eilučių prieš
dabartinę. Jei eilučių yra daugiau nei ekrane, programa sustos po kiekvieno
puslapio.

[#],[#],#m - PERKELTI BLOKĄ (MOVE)

Perkelia teksto bloką prieš trečiajame parametre nurodytą eilutę. Tai veikia
kaip kopijavimas ir vėlesnis originalo ištrynimas.

[#][,#]p - PUSLAPIS (PAGE)

Panašu į l, tačiau numatytasis puslapis pradedamas nuo dabartinės eilutės, o
ne 11 eilučių prieš ją.

q - IŠEITI (QUIT)

Savaime suprantama. Išeina neišsaugojus papildomų pakeitimų.

[#][,#][?]r$,$ - PAKEISTI EILUTĘ (REPLACE)

Pakeičia visus pirmosios eilutės pasikartojimus antrąja nurodytame diapazone.
Be pirmojo parametro pradedama po dabartinės eilutės; be antrojo – baigiama
failo gale.

Klaustukas (?) reikalauja patvirtinimo prieš kiekvieną pakeitimą.

[#][,#][?]s$ - IEŠKOTI EILUTĖS (SEARCH)

Ieško pirmosios eilutės, kurioje yra nurodyta seka. Be pirmojo parametro
ieškoma nuo dabartinės eilutės, be antrojo – iki galo.

Klaustukas (?) įjungia patvirtinimą. Atsakius „N“ (Ne), paieška tęsiama;
atsakius „T“ (Taip), ji nutraukiama. Sėkmės atveju dabartinė eilutė
nustatoma ties radimo vieta.

[#]t failas - PERKELTI FAILĄ (TRANSFER)

Įterpia išorinio failo turinį prieš nurodytą eilutę. Jei nenurodyta,
naudojama dabartinė eilutė.

[#]w failas - ĮRAŠYTI FAILĄ (WRITE)

Įrašo nurodytą eilučių skaičių į failą. Be parametrų įrašomas visas buferis.

Commande : unformat

  UNFORMAT tente de restaurer un disque qui a été formaté.

Syntaxe :

  UNFORMAT lecteur [/J]
  UNFORMAT lecteur [/U] [/L] [/TEST] [/P]
  UNFORMAT [/PARTN] [/L]
  UNFORMAT [/?]
           lecteur :  La lettre du lecteur à restaurer, par exemple C:

Options :

  /J      Vérifie que les fichiers miroirs du lecteur sont synchronisés
          avec le disque.
  /U      * Restaure sans utiliser les fichiers MIRROR.
  /L      Liste tous les fichiers et dossiers trouvés, ou, lors d'une
          utilisation avec l'option /PARTN, affiche les tables de
          partition actuelles.
  /TEST   * Affiche les informations mais n'écrit pas les modifications
         sur le disque.
  /P      * Envoie les messages de sortie vers LPT1.
  /PARTN  Restaure les tables de partition du disque qui ont été
          sauvegardées avec MIRROR /PARTN.
  /?      Affiche l'aide.
  *       Indique des fonctions qui ne sont pas encore disponibles.

Commentaires :

  Les informations sur MIRROR peuvent être utiles pour unformat.

Exemples :

  UNFORMAT c: /J
  UNFORMAT c: /U /L /TEST /P
  UNFORMAT /PARTN /L

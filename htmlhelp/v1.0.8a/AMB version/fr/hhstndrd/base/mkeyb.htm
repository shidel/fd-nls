Commande : mkeyb

  MKEYB est un pilote de clavier compact.

Syntaxe :

  mkeyb [options] code
  keyb [options] code

  Remappe les touches conformément à ce qui est nécessaire dans certains
  pays. Utilise approximativement 500 octets (en fonction du pays).


  code    Emplacement            Page de code requise
  ===================================================
   BE     Belgique
   BR     Brésil
   BX     Brésil (étendu)
   DK     Danemark
   FR     France
   GR     Allemagne
   GRC    Allemagne (avec des COMBIs)
   HE     Hébreu
   IT     Italie
   LA     Amérique Latine
   NL     Pays-Bas
   NO     Norvège
   PL     Pologne
   PO     Portugal
   RU     Russie                         866
   SF     Suisse (français)
   SG     Suisse (allemand)
   SP     Espagne                        850
   SU     Finlande
   SV     Suède
   UK     Royaume-Uni

Options :

  /l  Liste tous les claviers disponibles.
  /u  Décharge de la mémoire le pilote de clavier.
  /t  Teste le pilote de clavier spécifié, ne se transforme pas en
      résident.
  /?  Affiche l'aide.

Commentaires :

  MKEYB (KEYB) se charge automatiquement aussi haut que possible
  - il n'est donc pas nécessaire d'utiliser LH / LOADHIGH.
  MKEYB (KEYB) est fourni avec le paquet MKEYB040.

Exemples :

  mkeyb GR     charge le pilote de clavier allemand
  mkeyb /u     décharge le pilote de clavier
  mkeyb GR /t  teste le clavier allemand, ne se transforme pas en
               résident

Mise en réseau de FreeDOS - 3Com

  Si votre NIC 3Com n'est pas prise en charge par les pilotes de paquets
  Crynwr vous pouvez trouver un
  pilote ici :
  ftp://ftp.3com.com/pub/nic/3c90x/3c90x2.exe (*01)
  Il s'agit d'EtherCD v5.4, second disque sur 2 de la famille
  d'adaptateurs 3C90x. Le second disque contient les pilotes DOS ODI
  et NDIS, le pilote de paquets, les fichiers d'aide et les diagnostics
  DOS. C'est un exécutable DOS auto-extractible. Après l'avoir exécuté,
  allez dans le sous-répertoire PKTDVR et lancez 3C90XPD.COM. Cela
  devrait détecter automatiquement votre NIC, ce qui peut prendre du
  temps, et installer le pilote de paquets dans le premier vecteur
  d'interruption disponible, commençant par 0x60. Le vecteur
  d'interruption peut aussi être spécifié avec /I; voyez /? pour les
  détails. Vous devez ajouter cela à votre AUTOEXEC.BAT.

  Par la suite, vous pouvez configurer et utiliser toute application
  WatTCP telle que SSH (cf. SSH2DOS) et Arachne. Utilisez ce qui suit
  dans votre WATTCP.CFG pour spécifier le vecteur d'interruption :
    pkt.vector = 0x60

  (*01) internet: "3c90x2.exe"

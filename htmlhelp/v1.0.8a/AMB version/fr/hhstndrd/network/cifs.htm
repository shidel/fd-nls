Mise en réseau de FreeDOS - CIFS (alias NetBEUI)

  Common Internet File System (CIFS) est une méthode pour deux ou
  davantage d'ordinateurs de partager des données sur un réseau. Il est
  dérivé du protocole Server Message Block (SMB), qui est dérivé de
  NetBIOS.
  NetBIOS était à l'origine une extension de l'API du BIOS PC. Il a été
  étendu pour d'autres utilisations et de multiples systèmes d'exploitation
  par Microsoft et d'autres. Microsoft a fourni un client pour DOS, mais
  depuis il a disparu et est réapparu. Jacco de Leeuw fournit des détails
  sur les logiciels de Réseau pour DOS, où les trouver et comment les
  installer.
  Malheureusement, jusqu'à présent un client Samba libre n'a pas été
  développé pour DOS et alors que le client MS fonctionne, il n'est pas
  très compatible avec d'autres systèmes CIFS. Voyez les références
  ci-dessous pour davantage de détails.

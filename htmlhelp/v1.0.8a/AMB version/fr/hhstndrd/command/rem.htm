Commande : rem

  REM enregistre des commentaires (remarques) dans un fichier batch
  ou dans CONFIG.SYS.
  REM est une commande de FICHIER BATCH / AUTOEXEC.BAT.

Syntaxe :

  REM [commentaire]

Options :

  commentaire tout commentaire que vous voudriez ajouter pour vous
              souvenir de pourquoi vous avez fait ceci ou cela ; cela
              peut aussi être une commande qui ne doit pas être
              exécutée (peut-être pour des tests).

Commentaires :

  FreeDOS ignore tout ce qui se trouve sur une ligne REM.
  Utilisez les remarques pour ajouter des notes explicatives dans
  config.sys, et pour désactiver des lignes dans le fichier config.sys
  afin qu'elles ne s'exécutent pas.
  Dans CONFIG.SYS vous pouvez aussi utiliser ";" au début d'une ligne.
  REM est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner. REM existe aussi en tant que commande
  CONFIG.SYS / FDCONFIG.SYS.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT :
    REM Définir la dernière lettre de lecteur disponible :
    lastdrive=z
    REM Je ne veux plus utiliser ce pilote :
    REM DEVICE=FOO.SYS

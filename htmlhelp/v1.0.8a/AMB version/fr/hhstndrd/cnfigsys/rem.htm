Commande : rem

  REM enregistre des commentaires (remarques) dans un fichier batch ou
  dans CONFIG.SYS / FDCONFIG.SYS. ';' a le même effet que 'REM'.

Syntaxe :

  REM [commentaire]
  ; [commentaire]

Options :

  commentaire  tout commentaire que vous voudriez ajouter pour vous
               souvenir de pourquoi vous avez fait ceci ou cela ; peut
               aussi être une commande qui ne doit pas être exécutée
               (peut-être pour des tests).

Comments:

  FreeDOS ignorera tout ce qui se trouve sur une ligne REM.
  Utilisez des remarques (REM) pour ajouter des notes explicatives à
  config.sys et pour désactiver des lignes dans le fichier config.sys
  afin d'empêcher qu'elles ne soient exécutées.
  Dans CONFIG.SYS vous pouvez aussi utiliser ";" au début d'une ligne.
  REM est une commande interne à kernel.sys et ne nécessite aucun
  autre fichier pour fonctionner.
  REM existe aussi en tant que commande AUTOEXEC.BAT/BATCH/COMMAND.COM.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    REM DOS=HIGH,UMB  a le même effet !
    ;DOS=HIGH,UMB     a le même effet !

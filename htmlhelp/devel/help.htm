Command: help / htmlhelp

  HELP / HTMLHELP is a browser and a collection of FreeDOS help files in
  html format. Meanwhile, the help files exist in several languages.
  Click here if you want to know how to use the help.

Syntax:

  HELP [options] [topic] OR:
  HTMLHELP [options] [topic]

Options:

  Basic Options:
  topic     Show help on this topic.
  /M        Force monochrome display.
  /F1       Use fancy color scheme 1.
  /F2       Use fancy color scheme 2.
  /A        Uses ASCII instead of extended characters.
  /Cnnn     Tell HELP the codepage is nnn, rather than let detect.
  /?        Shows the help.
  Advanced Options:
  /O[path[file]] Overrides the help path.
            If no file is specified, index.htm is assumed.
  /Lfile    Loads a file other than index.htm relative to the help path.
  /Htopic   When the user presses F1 or clicks on "help on help" help
            will load this file. Default is help.htm.
  Environment Variables:
  HELPPATH  Directory that contains your help files.
  HELPCMD   Put /M, /A, F1, F2 here to make them default.

Comments:

  The collection of help files is also named HHSTNDRD.
  Current version: HHSTNDRD 1.1.0 en.
  In FreeDOS (not in internet) HELP consists of a DOS HTML viewer
  (htmlhelp.exe / help.exe) and the HTML help files. In FreeDOS the help
  files are zipped. On the Internet only the unzipped htm files are in use.
  HTMLHELP / HELP supports NLS (national language support).
  If the local NLS file does not exist, HELP / HTMLHELP falls
  back to English language.

Examples:

  HTMLHELP /LC:\pathxy\append.htm

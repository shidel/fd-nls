Utilities, Apps, Archivers, Editors, Unix-like tools

  These programs are not a part of the standard FreeDOS distribution, but
  of course they work with FreeDOS. To make them work you need an existing
  executable file (*.com or *.exe file) and FreeDOS must already be booted.
  You can execute them by typing the name, e.g. "tee" or "foxcalc".
  The program should be in the actual directory, otherwise you should use
  the CD or PATH command before.
  In this section you find only a restricted amount of tools which are
  often used together with FreeDOS. If you need a special tool please go
  to the internet and have a look there.
  This section contains a mixture of Utilities, Apps, Archivers, Editors
  and Unix-like tools. This makes it easier to find programs by their
  name; if they were splitted, you would have to look at a lot of
  different positions.

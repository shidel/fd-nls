External commands:

  These files are part of the standard FreeDOS distribution.
  To make them work you need an existing executable file (*.com or *.exe
  file) and FreeDOS must already be booted.
  You can execute them by typing the name, e.g. "edit" or "format" and -
  if necessary - the option / switch and so on, see SYNTAX.
  The program should be in the actual directory, otherwise you should use
  the CD or PATH command before.

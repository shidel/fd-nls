Comment utiliser l'aide de FreeDOS (hhstndrd 1.0.7a fr)

  Avec la SOURIS:
  * Les liens des sujets sont surlignés en jaune (dans l'aide de FreeDOS).
    Sur Internet ils sont marqués comme d'habitude.
  * Cliquez sur un lien pour voir davantage sur ce sujet.
  * Cliquez sur l'ascenseur pour faire défiler la page vers le haut ou le bas.
  * Quittez l'aide en cliquant sur "Sortie" en haut de l'écran. (dans l'aide
    de FreeDOS)
  * Davantage d'options de navigation fournies en bas de l'écran.

  Avec le CLAVIER (dans l'aie de FreeDOS - pas sur Internet) :
  * Utilisez les touches de curseur pour surligner un sujet, puis appuyez sur
    Entrée.
  * Page Up et Page Down font défiler les informations d'aide.
  * Alt+X quitte l'aide. (Alt+F4 fonctionne également)
  * Alt+Curseur gauche remonte un sujet. (Échap, Alt+B and Retour arrière
    fonctionnent aussi)
  * Alt+Curseur droite va sur le sujet suivant. (Alt+F fonctionne également)
  * Alt+C retourne à la page de contenu.
  * F1 affiche cet écran d'aide.
  * Il existe d'autres moyens de faire défiler les informations d'aide sur
    l'écran. Home et Fin vont au tout début ou à la fin du document.
    Maintenir Alt tout en appuyant sur les touches de curseur haut et bas fait
    défiler l'aide progressivement.

  Comprendre la "syntaxe" des rubriques d'aide :
    Sur la plupart des rubriques d'aide, vous verrez quelque chose comme :

      commande obligatoire [optionnel] [/commutateur] [/A|/B]

  Voici une explication des différentes parties de cette ligne :
    commande
  C'est le nom de la commande.
    obligatoire
  Tout ce qui suit le nom de la commande, qui n'est pas entre crochets
  [comme ici], est un paramètre obligatoire.
  La commande indiquera une erreur si vous ne fournissez pas cette information.
    [optionnel]
  les options sont écrites entre crochets, [comme ici]. Elles modifient
  la manière dont la commande s'exécute. Les crochets ne sont pas destinés à
  être saisis lorsque la commande est utilisée.
    [/commutateur]
  Les optins qui commencent avec un / sont appelées des commutateurs. Le
  symbole / est en fait tapé lorsque le commutateur est utilisé.
    [/A|/B]
  Lorsqu'il y a le symbole | , cela signifie qu'il existe un choix entre
  deux options. Soit /A soit /B est autorisé, mais pas les deux à la fois.
  Le | n'est pas destiné à être saisi lorsque la commande est utilisée.

  Autres façons d'obtenir de l'aide :
  1. La plupart des commandes décriront leurs fonctionnalités et options en
     utilisant /? sur elles sur la ligne de commande. Par exemple :
     dir /?
  2. La commande FASTHELP (FDHELP ou FSUITE04). C'est un clone de fasthelp de
     MS-DOS. Si vous ne trouvez pas ce que vous recherchez dans l'aide, il
     peut être présent dans fasthelp, ou vice versa.
  3. Les commandes WHATIS et APROPOS.
  4. La commande HELP (HTMLHELP).
  5. Le sous-répertoire doc dans le répertoire DOS contient de nombreux manuels
     détaillés pour divers programmes FreeDOS.
  6. Visitez le site Internet de FreeDOS, http://www.freedos.org

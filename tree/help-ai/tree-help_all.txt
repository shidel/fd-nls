Este é o arquivo readme para o programa TREE do FreeDOS.

Exibe graficamente a estrutura de pastas de uma unidade ou caminho.

Uso:  TREE [/A] [/F] [CAMINHO]
  /A    Usa caracteres ASCII para desenho de linhas.
  /F    Exibe arquivos nos diretórios (em vez de apenas diretórios).
  CAMINHO Onde começar a exibir a árvore de diretórios, o padrão é o
        diretório atual.

Este programa tem suporte internacional usando a biblioteca "Cats"
(implementação catopen/catgets para DOS). Você pode baixar "Cats" de
http://www.freedos.org/jhall/cats
